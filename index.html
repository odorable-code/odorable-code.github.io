02. PROJECT 01 (1인 단독 개발)
RAG 기반 고민 상담 캐릭터 챗봇 메이티 (Matey)
개발 기간: 2026.05 ~ 2026.06 (개인 프로젝트)
주요 역할: 기획, 아키텍처 설계, React/SpringBoot/FastAPI 및 배포 인프라 1인 단독 개발
프론트엔드(React), 백엔드(Spring Boot), AI(FastAPI/LLM)를 유기적으로 조율하여 대화형 심리 및 일상 고민 상담 웹서비스의 전체 파이프라인을 혼자서 완성해 낸 대표적인 웹 풀스택 프로젝트입니다.
  • 독자적인 3-Tier 아키텍처 설계: 사용자 인증 및 계정 처리는 Spring Boot로, 임베딩 및 RAG LLM 연산은 FastAPI 서버로 비동기 중계함으로써 연산 부하를 완벽히 격리한 구조를 직접 설계했습니다.
  • React UI/UX 구현: 대화 말풍선 컴포넌트, 회원가입/로그인 세션 처리 및 상담 내역 조회 마이페이지 뷰를 React를 사용해 백엔드 API 규격에 맞춰 독자 개발했습니다.
  • FastAPI RAG AI 파이프라인: 사용자 발화를 임베딩한 뒤 Chroma DB에서 관련 캐릭터 페르소나 정보를 탐색하고, 이를 시스템 프롬프트와 조합해 Gemini 3.1 Flash API로 최적의 답변을 생성하는 파이프라인을 단독 완성했습니다.

03. FULL-STACK OPTIMIZATION
Matey 종합적 최적화 및 문제 해결
보안 및 무결성: 데이터 생명주기 관리 및 프라이버시 보호
인프라: 멀티 컨테이너 Docker 배포 환경
"백엔드의 데이터 제약과 프론트엔드의 세션 동기화, 클라우드 리소스 한계를 동시에 해결하는 풀스택 관점의 해결력"
  • 상담 요약 테이블 기반 대화 연속성: 프라이버시 보호를 위해 대화 원문은 자동/수동 영구 삭제 정책을 타게 설계했으나, 챗봇이 과거 맥락을 기억하도록 핵심 요약 테이블을 별도로 격리 구축해 화면단에는 변함없는 친밀한 대화 경험을 전달했습니다.
  • 프론트-백 연계 회원 탈퇴 무결성 해결: React에서 회원 탈퇴 시 백엔드 외래키 에러로 중단되던 에러를 MySQL DB에 ON DELETE CASCADE 마이그레이션 스크립트를 작성 적용하고, React 단에서 세션 제거 및 리다이렉트 흐름까지 일사천리로 동기화했습니다.
⚡ AWS EC2 프리티어 메모리 OOM 해결
GitHub Actions를 활용해 EC2에 Spring Boot와 FastAPI 도커 컨테이너를 함께 배포할 때 물리 메모리(1GB) 고갈로 빌드가 뻗어버리는 문제가 있었습니다. Linux 가상 Swap File(2GB) 설정을 적용하여 추가 비용 청구 없이 멀티 서버를 안정적으로 배포했습니다.
⚡ 페르소나 이탈 및 대화 꼬임 현상 해결
상담이 길어지면 LLM의 문맥 제약으로 캐릭터 페르소나가 무너지는 문제를, 데이터베이스 요약본 및 세밀한 시스템 프롬프트 가중치 조절을 통해 극복하여 일관된 챗봇 답변을 보장했습니다.
04. PROJECT 02 (FULL-STACK)
병원 실시간 예약 및 약국 검색 바로닥큐 (BaroDocQ)
개발 기간: 2026.03 ~ 2026.04 (4인 풀스택 프로젝트)
주요 역할: 실시간 알림 웹소켓 파이프라인 구현, 공공데이터 수집 스크립트 구축
공공데이터 API로 수집한 전국 병원/약국 데이터 적재를 기초로, 실시간 웹소켓 양방향 예약을 연동한 풀스택 웹 어플리케이션입니다.
  • WebSocket & STOMP 양방향 실시간 알림 파이프라인: 예약 상태 변화 시 사용자와 관리자(병원)가 즉각 정보를 받도록 Spring Boot WebSocket STOMP 브로커와 프론트엔드의 StompJS 클라이언트 모듈을 맞물려 개발함으로써, 실시간 UI 렌더링 동기화를 완수했습니다.
  • React 화면단 UI 컴포넌트 개발: 회원 가입/수정, 실시간 예약 신청 및 내역 조회, 병원 즐겨찾기, 병원 리뷰 작성 등 마이페이지와 핵심 예약 플로우의 프론트엔드 UI 컴포넌트를 직접 구현했습니다.
  • 데이터 수집 & DB 이관 스크립트 작성: 건강보험심사평가원 및 공공포털 API 데이터를 수집하는 Python 스크래퍼를 구축하고, 정제되지 않은 Null 값 처리 및 중복을 필터링해 MySQL에 대량 적재하는 End-to-End 파이프라인을 완성했습니다.

  • 공백기를 도약으로 이끈 습득력: 개발 직무 복귀를 위해 학점은행제 정보통신학사를 취득하고 패스트캠퍼스 풀스택 양성과정에서 7개월간 현대적인 웹 기술 스택(Spring Boot, React, FastAPI, Chroma DB, RAG)을 빠르게 습득하여 1인 서비스 구축으로 직접 증명했습니다.
